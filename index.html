مساعد نشر وتحديث المتجر على GitHub

نظام المزامنة الذاتية الخالي من التعقيد والتكاليف

عزيزتي آلاء، لتحديث المنتجات عند جميع زبائنكِ، يرجى تصدير الكود المحدث بالضغط على الأزرار أدناه، ثم الصاقه في ملفات products.json و categories.json في حساب كود GitHub الخاص بكِ.

خطوات التحديث البسيطة (خلال دقيقة واحدة):

  • اضغطي على زر تصدير كود المنتجات (سيتم نسخ الملف وتحميله تلقائياً).
  • اضغطي هنا: فتح مستودع GitHub الخاص بكِ .
  • ابحثي عن ملف products.json (إن لم يكن موجوداً، قومي بإنشائه بالضغط على Add file -> Create new file وكتابة الاسم بنفس الحروف).
  • اضغطي على أيقونة التعديل (القلم ✏️) بالجهة العليا للملف.
  • احذفي أي كود قديم، ثم الصقي الكود الجديد المنسوخ، واضغطي Commit changes بالأخضر لحفظ التعديل.
